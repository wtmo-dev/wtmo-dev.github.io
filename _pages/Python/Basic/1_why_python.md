# **파이썬**
---
다양한 프로그래밍 언어가 존재하지만 방대한 라이브러리와 유저 친화적인 언어로 높은 범용성을 지닌 언어로써 점점 인지도가 증가하며 미래가 더 밝은 언어이기도 하다. 파이썬의 강력한 장점은 아래와 같다.

* 객체 지향형 언어
* 오픈소스 프로그래밍 언어
* 인터프리터 언어
* 데이터를 다루는데 특화되어있음
* 높은 점유율
* AI 모델링에 적합함
